SharedPaint is an AGTk shared application. With this shared application we try to provide a basic example to develop applications on AGTk.

This software is released under GNU GPL.
Read the file 'COPYING' for more information.

Summary:
- SharedPaint currently runs on AGTk 3.0.2
- We use Eclipse (an open development platform, see www.eclipse.org) with PyDEV for Python programming.

____How to create a *.agpkg3 file____

You have to use ANT (http://ant.apache.org/)

Type:

[user@MainFrame]$ ant

Buildfile: build.xml

init:

modo_uso:
     [echo] -----------------------------------------
     [echo] ** Sharedpaint build file **
     [echo]    Version 1.2
     [echo] -----------------------------------------
     [echo] 
     [echo] Options :
     [echo] clean         - Clean /bin and /docs/API
     [echo] build_agpkg   - Build the Access Grid Package

Type: 

[user@MainFrame]$ ant build_agpkg

____How to create a HTML API____

You have to use Doxygen (http://www.stack.nl/~dimitri/doxygen/)

Type:

[user@MainFrame]$ doxygen buildAPI
