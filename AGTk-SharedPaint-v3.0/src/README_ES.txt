____Presentación____

SharedPaint es una aplicación compartida Access Grid que permite trabajar
sobre  una imagen de interés. Este  programa  ha sido desarrollado por el
grupo de investigación IMAGINE de la Universidad de los Andes 
<http://imagine.uniandes.edu.co>.

Copyright (C) Universidad de los Andes.

Este programa se publica bajo los términos de GNU GPL.
Para mayor información consulte el archivo 'COPYING'.


____Características____

* Carga una imagen almacenada en el servidor AG. Formatos soportados:
JPG, PNG, GIF y  TIFF.

* Actualiza la sesión de un participante nuevo.

* Registra del estado de una sesión.


____Instalación____

i.	Cargar el archivo 'SharedPaint-v3.0-agpkg3' en la sección 'Data'
	del servidor.

ii.	En la sección 'Data', sobre el archivo 'SharedPaint-v3.0-agpkg3', 
	ejecutar 'Click derecho -> Open'.
